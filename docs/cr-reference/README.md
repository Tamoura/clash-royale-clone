# CR reference assets

Official Clash Royale card art collected as **visual reference only**
to guide our original low-poly 3D character models. These Supercell
images are **not** used in the build or shipped — the game renders
all-original primitives (see top-level `README.md`).

- `cards/` — official card art for all 25 deck cards (`<slug>.png`).
- `character-design.md` — per-character design breakdown (silhouette,
  palette, signature features, 3D modeling cues).

Source: Clash Royale Fandom wiki via its MediaWiki API. To refresh,
see the "Source / refresh" section in `character-design.md`.
