# Tombstone (spawner building)

3-elixir building that never attacks (damage 0, range 0) and drips
skeleton waves every 6s for its 30s lifetime. Pure card definition —
the spawner mechanic (spawnUnitId/spawnInterval) already flowed
through spawnBuilding since the Witch round.

- 3D: gravestone (mound, slab, rounded cap, inscription, tiny skull,
  green necro orb). 2D HUD art matches.
- README deck count updated to 21.
- Future: CR's tombstone bursts 4 skeletons on death — would need a
  deathSpawn mechanic; noted, not built.
