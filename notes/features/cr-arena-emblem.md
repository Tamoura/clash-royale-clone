# CR arena emblem + pink elixir (real-screenshot round)

User shared 10 real CR battle screenshots. Two universal gaps:
1. The iconic large GOLDEN STAR emblem inlaid on each player's
   half (in every screenshot) — we had none.
2. Elixir economy is hot PINK/magenta in CR, ours was purple.

- makeStarEmblem(z): flat 6-pointed ShapeGeometry star (gold toon
  material + grain) with a darker center ring, one per half at
  z = ±(ARENA_HEIGHT/4 + 0.5).
- Elixir CSS shifted purple -> CR hot pink (fill gradient, droplet
  badges, cost-droplet tip): #f23aa8 / #ff7ac6 / #c01878.
